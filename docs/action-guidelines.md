# 초기 조치 권고 가이드

이 문서는 정교한 현장 매뉴얼이 준비되기 전까지 사용할 초기 조치 권고 기준 초안이다. 실제 운영 적용 전에는 안전 담당 부서 검토가 필요하다.

## 공통 원칙

- 실제 위험이 의심되면 현장 재확인을 우선 권고한다.
- 즉시 위험 가능성이 있으면 접근 통제 또는 작업 중지를 우선 고려한다.
- 권고 문구는 과도한 단정 대신 확인과 조치를 함께 제시한다.

## 유형별 권고 초안

### 화재/불꽃

- 즉시 현장 확인
- 필요 시 작업 중지 및 접근 통제
- 소화 설비 확인 및 안전 담당자 보고

### 연기/그을음

- 발열 또는 화재 징후 여부 현장 점검
- 전기/설비 이상 여부 확인
- 필요 시 주변 작업 일시 중지

### 누수/액체 유출

- 누수 원인 또는 유출 지점 확인
- 미끄럼 위험 구간 통제 또는 표지 설치
- 바닥 정리 및 설비 이상 여부 점검

### 파손/균열/손상

- 손상 부위 현장 확인
- 사용 중 설비라면 사용 지속 가능 여부 점검
- 필요 시 보수 요청 또는 사용 제한

### 장애물 적치/통로 방해

- 통로 확보 여부 확인
- 적치물 즉시 정리 권고
- 재발 방지를 위한 적치 기준 점검

### 설비 위치 변경/이탈

- 원위치 여부 확인
- 설비 고정 상태 및 작업 영향 점검
- 필요 시 표준 배치 상태로 복구

### 오염/청결 불량

- 오염 원인 확인
- 청소 및 주변 정리 수행
- 반복 발생 시 관리 주기 점검

### 정리정돈 미흡

- 작업 구역 정리 상태 확인
- 물품 분류/정위치 배치 권고
- 관리 기준 준수 여부 점검

### 판단 불가

- 추가 이미지 또는 현장 재촬영 요청
- 사람 검토 대상으로 분류

## 권고 문장 스타일 가이드

- 짧고 실행 가능하게 작성한다.
- `확인`, `점검`, `정리`, `보고`, `통제` 같은 동사를 우선 사용한다.
- 모델이 단독으로 안전 판정을 확정하지 않도록 한다.
